使用场景绘图

终于到OneKeyD2D作为窗口库的特色部分了

场景或者说房间绘图是为了模块化的管理，和便捷的编码
之前的窗口绘图总是要设置什么消息回调，要适配动态的UI（可以使用函数固定窗口）
而且好不容易写好一个窗口布局后会发现需要切换界面时又要把之前的操作隔离，要把按钮解绑，只是一个游戏的开始界面跳转到游玩界面就有一大堆的东西要操作，要注意冲突，看着听着就让人头大

现在，使用场景绘图，可以专心致志的绘制单个界面，布局场景时只需要对自己的布局负责，所有的切换与管理交由系统并且
提供基类函数

在进入场景时调用
Start()

在每一帧调用
Update(float 上一帧绘制的间隔时间,单位毫秒)

离开场景时调用
End()

让你的编程体验如Unity一般便捷

内容

  • 定义场景
  • 添加场景对象
  • 初始化场景对象
  • 使用场景管理器

定义一个场景

使用继承的方式定义场景类

每个场景类都允许单独设置用户数据，也允许定义场景的消息回调，但是窗口的用户数据会被占用，窗口的用户数据会被改为Room指针，如果进行修改，会导致未知错误

新的文件要记得修改编码格式为UTF-8

#include<OneKeyD2D/OneKeyD2D.hpp>
using namespace Game;
class FirstRoom:public Room
{
};

好了，现在就有一个场景了，不过什么都没有，也无法发挥作用

添加场景对象

所有场景对象都在命名空间Game::RoomObj中

场景的一切都是由场景对象构成，场景对象有UI对象与Scence对象两部分
他们的坐标系不同，UI对象直接使用窗口坐标系，Scence对象使用场景坐标系，可以通过调整每个场景中自带的摄像机或直接对对象进行调整
场景对象一般是不包含资源的，所以要绑定资源，

这里先定义构造函数用于初始化

定义一个图片就要使用两个变量可能会显得比较繁琐，有两种解决方案

  • 使用ImageUIPack定义背景图片，这里面已经包含了d2dPicture类型的公共变量m_Data，并且已经与ImageUI绑定，只需要加载便可以设置了，但是当有多个场景使用相同的资源时，会造成内存浪费

  • 利用每个房间的UserData，定义一个结构体存储数据，将结构体指针放入UserData，初始化时绑定到对象（一个数据允许绑定多个对象），让一个资源可以反复使用

    //设置宽度为窗口的100%，100%

    //m_Background.SetShowWide(800, 600);

    //添加到场景对象列表，
    // 默认添加位置为-1，也就是倒数第一个，
    //添加位置表示绘制顺序
    //如果没有添加就不会绘制
    AddObject(m_Background);
    //初始化按钮
    m_Button.Init(
        window,//用于传递d2d设备的窗口
        L"切换场景",//按钮显示的文本
        nullptr,//按钮回调，由于场景2还没有写，所以先设置为空

        this//将场景指针作为参数传递
    );
    //设置按钮绘制位置所在矩形
    //这里仍然使用窗口百分比坐标
    m_Button.SetShowRectForUI(
        D2D1::RectF(
            0.4, 0.5,
            0.6, 0.6
        )
    );
    //添加到对象列表，让系统管理运行状态
    AddObject(m_Button);
}

使用场景管理器

利用场景管理器来显式的发送场景切换指令

首先在主函数中定义一个场景管理器变量

int main()
{
	//初始化COM设备
	AutoInit autoInit;
	//窗口索引标识
	MainWind_D2D Window;
	//场景管理器需要与窗口绑定，
	//所以要在窗口变量后创建，
	// 运行创建在窗口创建函数之前
	RoomManage roomManage(Window);
	//创建窗口
	Window.CreateWind();
    //设置标题
	Window.SetText(L"OneKeyD2D场景教学");
	//由于对象初始化需要用到d2d设备，
	//所以必须在窗口创建后创建
	FirstRoom room1(&Window);
	//将场景交由场景管理器管理，并为器分配独有的id
	//如果id已被占用不会覆盖原场景，并返回-1
	roomManage.AddRoom(room1, 1);
	//跳转到指定场景id的场景
	//如果什么都没有指定则为什么都没有的黑窗口
	roomManage.GotoRoom(1);
	//开启消息循环
	Window.StaticWindowMessageLoop();
	return 0;
}

现在就能正确绘制了，并且ui会随着窗口大小的改变自动调整

然后在每一帧都检测按键情况，
使用GetKeyState函数检测按键状态，参数为虚拟键键码
比如实现当按下空格时按钮会发生抖动1秒

//设置抖动的时间为1秒
const float timingJitter = 1000.f;
//用于检测当前抖动运行时间
static float RunTime = 0;

//检测是否按下空格，如果按下则返回值小于0
if (GetKeyState(VK_SPACE) < 0)
{
    //设置运行时间为0，表示开始抖动
    RunTime = 0;
}		
//如果运行时间小于抖动时间，表示要继续抖动
if (RunTime < timingJitter)
{
    //加上运行时间
    RunTime += IntervalTime;

    //SetShowPositionForUI是跟随窗口大小调整设置的，
    //所以不会立即显示，因此要使用SetShowPosition

    //在激活的场景中可以通过调用基类函数GetRoomManage来获取
    //场景管理器，然后通过场景管理器获得当前窗口,进而读取窗口大小
    auto WindowSize = GetRoomManage()->GetCurrentWindow()->GetWindSize();
    //计算在当前时间抖动后的位置
    auto NewPos = Vector::Vec2(
        WindowSize.cx * 0.4, 
        WindowSize.cy * 0.5 + std::sin(RunTime) * 30);
    m_Button.SetShowPosition(NewPos.x, NewPos.y);
    //抖动动画结束后回到原位
    if (RunTime >= timingJitter)
    {
        m_Button.SetShowPosition(WindowSize.cx * 0.4, WindowSize.cy * 0.5);
    }
    
}

定义回调

//定义按键回调
static void KeyCallback(Room* roomP,int key,int,KeyMode mode)
{
	//将room从基类转换为当前类型
	auto* room2 = roomP->ToRoom<SecondRoom>();
	//确定消息为esc的字符消息
	if (mode == KM_CHAR&&key==VK_ESCAPE)
	{
		//跳转到场景1，基类中的GoToRoom函数是不能使用的，
		//他是给场景管理器调用用于初始化的，
		//切换场景使用场景管理器的GoToRoom
		room2->GetRoomManage()->GotoRoom(1);
	}
}

定义后使用基类函数
SetRoomKeyCallback设置场景按键回调

//设置按键回调

			}
			
		}
	}
	//设置按键回调
	static void GoToRoom2(MainWind* window,long long roomP)
	{
		auto* room = (FirstRoom*)roomP;
		room->GetRoomManage()->GotoRoom(2);

#pragma once
#include<OneKeyD2D/OneKeyD2D.hpp>
using namespace Game;
using namespace RoomObj;
class SecondRoom :public Game::Room
{
	//背景
	ImageUIPack m_Background;
	//定义按键回调
	static void KeyCallback(Room* roomP,int key,int,KeyMode mode)
	{
		//将room从基类转换为当前类型
		auto* room2 = roomP->ToRoom<SecondRoom>();
		//确定消息为esc的字符消息
		if (mode == KM_CHAR&&key==VK_ESCAPE)
		{
			//跳转到场景1，基类中的GoToRoom函数是不能使用的，
			//他是给场景管理器调用用于初始化的，
			//切换场景使用场景管理器的GoToRoom
			room2->GetRoomManage()->GotoRoom(1);
		}
	}
public:
	//初始化
	SecondRoom(MainWind_D2D* window)
	{
		//设置按键回调
		SetRoomKeyCallback(KeyCallback);
		//加载背景图片，（此图片由ai生成做演示）
		m_Background.m_Data.LoadPictureFromFile(L"背景2.png", window);
		//设置显示大小
		m_Background.SetShowWideForUI(1, 1);
		//添加到场景
		AddObject(m_Background);
	}
};
