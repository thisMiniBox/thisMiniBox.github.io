OneKeyD2D开发文档

编辑版本0.22

简介

OneKeyD2D是一个纯头文件的开源窗口库，封装了win32 gdi（简单封装）窗口和d2d（Direct2D，之后全部简称d2d）窗口的创建与使用。

采用元素的方法分类窗口组件，模块化的编程。d2d的绘制与传统的win32 gdi相比效率更高、更稳定

相比其它窗口库，OneKeyD2D有了场景坐标、摄像机坐标、窗口坐标的概念，实现各页面的随心切换与ui的自动管理，并提供瓦片地图、柏林噪声工具，帮助初学者快速制作简单的游戏。

创建窗口

之后都使用vs做示例

创建单个空窗口

首先引用头文件

#include<OneKeyD2D/OneKeyD2D.hpp>

这个头文件会以正确的顺序引用所有头文件，推荐使用这个，安全并且方便
如果比较介意全部引用，可以使用

#include<OneKeyD2D/MainWind.hpp>

这样只会引用基础库 MainWind.hpp

使用命名空间Game，所有的组件都在此命名空间中

using namespace Game;

现在可以直接创建一个d2d窗口

int main()
{
	//窗口索引标识
	MainWind_D2D window;
	//创建窗口
	window.CreateWind();
	//开启消息循环
	window.StaticWindowMessageLoop();
	return 0;
}

现在试试多窗口及嵌套窗口

先使用AutoClearWindBackground函数来实现窗口背景的切换和区分不同的窗口
先设置之前创建的window背景为绿色（要在消息循环前调用）

window.AutoClearWindBackground(RGB(0,255,0));

再按照之前的方法创建窗口2并设置为黄色

//窗口索引标识
MainWind_D2D window,window2;
//创建窗口
window.CreateWind();
window2.CreateWind();

window.AutoClearWindBackground(RGB(0,255,0));
window2.AutoClearWindBackground(RGB(255, 255, 0));

//开启消息循环
window.StaticWindowMessageLoop();

运行后成功创建两个不同的窗口

现在试试父子窗口
因为子窗口会遮挡父窗口的画面，所有父窗口一般使用gdi窗口便足够了
使用MainWind_GDI创建gdi窗口

MainWind_GDI fWindow;
//创建窗口，需要在子窗口创建之前创建
fWindow.CreateWind();

然后将窗口句柄作为参数放入CreateWind函数

window.CreateWind(fWindow.GethWnd());
window2.CreateWind(fWindow.GethWnd());

使用GethWnd函数便可以获取以创建成功的窗口的窗口句柄
现在运行 为什么只有一种颜色，而且移动窗口时会闪烁？
因为当创建为子窗口时，会默认占用所有客户区，现在两个子窗口的显示区域完全重叠了，所以我们应该使用MoveWind函数来移动窗口位置，让他们各占用一半

//获取父窗口的大小
auto fsize = fWindow.GetWindSize();
//移动子窗口让他们分别到左右两边
window.MoveWind(0, 0, fsize.cx * 0.5, fsize.cy);
window2.MoveWind(fsize.cx * 0.5, 0, fsize.cx, fsize.cy);

现在的运行效果就对了

完整代码

#include<OneKeyD2D/OneKeyD2D.hpp>

using namespace Game;
int main()
{
	//窗口索引标识
	MainWind_D2D window,window2;
	MainWind_GDI fWindow;
	//创建窗口
	fWindow.CreateWind();
	window.CreateWind(fWindow.GethWnd());
	window2.CreateWind(fWindow.GethWnd());

	window.AutoClearWindBackground(RGB(0,255,0));
	window2.AutoClearWindBackground(RGB(255, 255, 0));

	//获取父窗口的大小
	auto fsize = fWindow.GetWindSize();
	//移动子窗口让他们分别到左右两边
	window.MoveWind(0, 0, fsize.cx * 0.5, fsize.cy);
	window2.MoveWind(fsize.cx * 0.5, 0, fsize.cx, fsize.cy);

	//开启消息循环
	window.StaticWindowMessageLoop();
	return 0;
}

MainWind.hpp

基础库，封装了窗口的创建以及基础绘制

内容

  • d2dClickDetection
    窗口的点击检测器，用于制作按钮
  • MainWind
    窗口基类，包含gdi与d2d窗口共有的函数与基础功能的实现。
    是纯虚类，不可直接使用
  • MainWind_GDI
    继承自MainWind，表示GDI窗口
  • MainWind_D2D
    继承自MainWind，表示D2D窗口
  • SimpleSound
    实现简单的音频播放，封装的是PlaySound函数，所以只能播放wav文件，且不支持暂停跳转
  • Sound
    封装自mciSendString，能满足大多数音频播放需求

  • SetWide()

    检测指定位置是否在检测区域

    设置点击回调

类MainWind_D2D

介绍

表示d2d窗口的索引，继承自MainWind包含d2d窗口的基本信息，一个MainWind_d2d变量只能与一个窗口绑定，所以CreateWind方法只有在没有与窗口关联的情况下有效

方法

  • GetD2DTargetP()
    获取当前d2d窗口的d2d绘制目标句柄
  • GetD2DFactoryP()
    获取当前d2d工厂
  • D2DColorFToGdiColor()
    由D2D1_COLOR_F类型转换为COLORREF类型
  • GdiColorToD2DColor()
    由COLORREF类型转换为D2D1_COLOR_F类型
  • SetKeyCallback()
    设置按键回调
  • SetPaintCallback()
    设置绘图回调
  • SetWindSizeCallback()
    设置窗口大小调整回调
  • GetKeyCallback()
    获取当前按键回调
  • GetPaintCallback() 获取当前绘图回调
  • GetWindSizeCallback()
    获取当前窗口大小调整回调
  • DrawGeometry()
    绘制多边形（与MainWind中的方法由差异的）
  • GetPaintIntervalTime()
    获取上次绘制的间隔时间
  • SetPenStyle()
    设置画笔样式
  • SetPenColor()
    设置画笔颜色

enum class PenStyle
{
    // 实线样式
    SolidLine = PS_SOLID,

    // 虚线样式
    DashedLine = PS_DASH,

    // 点线样式
    DotLine = PS_DOT,

    // 虚线点线样式
    DashDot = PS_DASHDOT,

    // 双点线样式
    DashDotDot = PS_DASHDOTDOT,

    // 空样式，不绘制轮廓

    // 内框样式

PM_ASYNC
